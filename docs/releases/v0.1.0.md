# DIPS v0.1.0

Initial public release of the Digital Identity Protection System.

## Summary

DIPS v0.1.0 introduces a local-first identity security platform for Windows and Linux with a production-style CLI, a SOC-style desktop dashboard, privacy-respecting reporting, and a modular analysis pipeline for exposure, phishing, browser risk, breach intelligence, threat intelligence, and event correlation.

## Highlights

- Cross-platform Python CLI with typed config, structured logging, JSON and HTML reporting, and watch mode.
- SOC-style PySide6 dashboard with risk score, threat panels, event timeline, alert correlation, severity analytics, and demo mode.
- Defensive scanner coverage for identity exposure, credential hygiene, local privacy risks, browser posture, and phishing indicators.
- Breach intelligence, threat intelligence, AI security analysis, plugin extensibility, and event timeline correlation.
- Public-facing launch assets including screenshots, example reports, release-ready packaging metadata, and curated documentation.

## Included Changes

- CLI:
  - `scan`, `watch`, `show-config`, `demo`, and dashboard launch workflows
- Dashboard:
  - overview panels, threat-intel views, risk analytics, timeline panels, and report loading
- Scanners and modules:
  - identity exposure, credential hygiene, privacy risk, browser audit, phishing, breach intelligence, threat intelligence, AI analysis, and plugin hooks
- Reporting:
  - redacted JSON and standalone HTML reports with scoring summaries and timeline context
- Documentation:
  - installation, quick start, architecture, plugin, report, troubleshooting, and branding guides
